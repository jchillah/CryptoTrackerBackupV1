# Contributing to CryptoTracker

Vielen Dank, dass du zum CryptoTracker-Projekt beitragen möchtest! Wir freuen uns über Verbesserungsvorschläge, Fehlerberichte und Pull Requests. Bitte beachte die folgenden Richtlinien, bevor du Beiträge einreichst:

## Code of Conduct

Bitte verhalte dich respektvoll und professionell. Wir erwarten von allen Mitwirkenden:
- Höflichkeit und Respekt
- Konstruktive Kritik
- Zusammenarbeit im Team

## So kannst du beitragen

1. **Forke das Repository:** Forke das Projekt und erstelle einen neuen Branch für deine Änderungen.
2. **Implementiere deine Änderungen:** Stelle sicher, dass dein Code dem bestehenden Stil folgt. Füge Kommentare und Tests hinzu, wenn möglich.
3. **Schreibe aussagekräftige Commit Messages:** Beschreibe, welche Änderungen du vorgenommen hast und warum.
4. **Erstelle einen Pull Request:** Reiche deinen Pull Request ein und beschreibe deine Änderungen detailliert. Wenn du Vorschläge zur Verbesserung hast, kannst du auch ein Issue eröffnen.

## Issues

Wenn du Änderungen oder neue Features vorschlägst, erstelle bitte ein Issue mit einer detaillierten Beschreibung der gewünschten Änderung. Dies hilft uns, die Anforderungen zu verstehen und Prioritäten zu setzen.

## Kontakt

Bei Fragen oder Anregungen kannst du dich auch direkt per E-Mail an **Jaychillah@gmail.com** wenden.

Vielen Dank für deine Unterstützung!
